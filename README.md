# Data Visualization and Storytelling

## Tool Used
Power BI Desktop

## Dataset
Superstore Dataset

## Visualizations Created
1. Sales by Category
2. Profit by Region
3. Monthly Sales Trend
4. Sales by Segment

## Key Insights
- Technology generated highest sales.
- West region was most profitable.
- Consumer segment contributed most revenue.
- Sales showed seasonal trends.

## Outcome
Created an interactive dashboard and learned data storytelling techniques.
